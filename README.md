## Адвент-бот «Угадай, что выкину»

Телеграм-бот на Python, который с 26 декабря по 11 января каждый день в 11:11 отправляет подписчикам подарок (текст, аудио, фото, видео или gif).

### Установка

- **1. Клонировать / открыть проект**
- **2. Установить зависимости**

```bash
cd /Users/ekaterinanigmatulina/Advent
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

- **3. Настроить конфиг**

Создать файл `config.py` на основе `config.example.py` и указать:

- **TELEGRAM_BOT_TOKEN**: токен бота от BotFather  
- **ADMIN_IDS**: список Telegram ID админов  
- **TIMEZONE**: например `"Europe/Moscow"`

### Запуск бота

```bash
python bot.py
```

Бот должен работать постоянно (на компьютере, сервере или хостинге), чтобы ежедневная рассылка в 11:11 выполнялась вовремя.

### Основные команды пользователя

- **/start**: рассказ о боте и кнопки «Подписаться» / «Отписаться»
- **/help**: вручную получить подарок за сегодня (если он есть)
- **/creators**: информация о создателях

### Админ-интерфейс

- **/admin_add**: запустить мастер добавления элемента в адвент-календарь
  - шаг 1 — дата в формате `ГГГГ-ММ-ДД`
  - шаг 2 — тип: `text`, `audio`, `photo`, `video`, `gif`
  - шаг 3 — сам контент:
    - для `text`: текст сообщения
    - для остальных типов: путь к файлу на сервере или URL
- **/cancel**: отменить добавление

Данные календаря хранятся в `advent_data.json`, подписчики — в `subscribers.json`.
